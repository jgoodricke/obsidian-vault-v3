## Vim Keybinds
- Word Motions
	- w -> Ctrl+Right
	- b -> Ctrl+Left
	- e -> Ctrl+Right, then Left
- Line Start/End
	- 0 or ^ -> Home
	- $ -> End
- File start/end
	- gg -> Ctrl+Home
	- G -> Ctrl+Endl
- Page movement
	- Ctrl-d
	- Ctrl-u
- Search
	- / -> F3
